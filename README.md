# STM32-Clearing_Robot

# 一、产品简介
本产品是一款基于STM32与OpenMV的智能视觉小车系统，融合嵌入式控制与机器视觉技术，实现对目标物体的自动识别、定位与跟随。系统通过OpenMV采集图像并提取目标坐标，经串口传输至STM32进行运动控制决策，驱动电机完成前进、转向等动作。整体结构紧凑、响应迅速，具备一定的环境适应能力，可广泛应用于智能巡检、自动跟随及服务机器人等场景。


# 二、应用场景
本产品可应用于多种智能化场景。在智能巡检领域，可用于替代人工完成固定路线或目标跟踪任务，提高效率与安全性；在服务机器人领域，可实现对用户或指定物体的自动跟随，提升交互体验；在仓储与物流场景中，可用于简单的物料搬运与目标引导；此外，在教学与科研中，也可作为嵌入式与机器视觉结合的实验平台，帮助学习相关技术原理与应用开发。


# 三、产品概述
本产品为STM32与OpenMV的智能扫地机器人系统。系统以OpenMV作为视觉感知核心，在OpenMV IDE环境下开发图像处理算法，实现了对地面垃圾的目标检测与识别。识别结果通过串口通信实时传输至STM32主控制器。在STM32端，使用Keil进行开发，通过PWM技术精确驱动电机，完成了小车的运动控制与自动清扫路径规划。

目前提供的1.0版本主要功能如下： 
1.视觉识别垃圾，并且清理
2.通过语音控制小车
3.通过4G通信模块，实现远程数据上传与状态监控

# 四、产品参数
  1、长宽高：23.5cm*17.5cm*7.4cm。
  2、开机状态下续航时长为1小时（选用12V锂电池）
  3、视觉识别垃圾最远70cm。
  4、控制RGB的范围可以实现多种颜色的垃圾识别
